# 교실 퀴즈 사용 설명서

같은 와이파이에 붙은 학생들이 **QR로 접속**해 함께 푸는 교실 실시간 퀴즈 게임 서버입니다.
교사 PC에서 **exe 하나를 더블클릭**하면 끝 — 학생은 앱 설치 없이 **브라우저**로 들어옵니다.

## 1. 소개

- 교사 PC에서 **`classroom-quiz.exe` 더블클릭** → 서버가 켜지고 기본 브라우저에 **교사 화면**이 자동으로 열립니다.
- 학생은 교사 화면의 **QR을 찍어** 닉네임만 입력하면 입장합니다(앱 설치 없음).
- 진행은 **로비 → 문제 → 정답 공개 → 순위표 → 시상대** 순서. 교사가 화면에서 버튼으로 넘깁니다.
- 인터넷이 필요 없습니다 — 웹 화면·QR이 exe 안에 들어 있어 **오프라인 교실 와이파이(LAN)** 에서도 동작합니다.
- 배포물은 **exe 파일 하나**. 만든 퀴즈는 **JSON 파일 하나**(그림까지 내장)라 다른 교사와 쉽게 주고받습니다.

## 2. 빠른 시작

1. `classroom-quiz.exe`를 **더블클릭**합니다. → 교사 화면(`/host`)이 자동으로 열립니다.
2. (처음이면) 화면 위 **＋ 퀴즈 만들기**로 문제를 만들고 **저장**합니다. → [5. 퀴즈 만들기](#5-퀴즈-만들기-편집기)
3. 학생들에게 화면의 **QR**을 보여줍니다(TV/프로젝터). 멀면 **전체화면으로 크게** 버튼을 누르세요.
4. 학생이 QR을 찍고 **닉네임**을 넣으면 교사 화면에 실시간으로 뜹니다.
5. 퀴즈를 고르고 **시작하기** → **정답 공개 → 순위표 → 다음 문제 → … → 최종 결과**.
6. 끝나면 결과가 자동 저장됩니다. **📊 결과**에서 보고 **CSV로 내보내기** 할 수 있습니다.

> 실수로 시작했거나 처음으로 돌아가려면 → 화면 왼쪽 위 **엠블럼을 클릭**하세요.

---

## 3. 화면 구성

브라우저 주소(같은 PC 기준)로 구분됩니다. **편집기·결과는 교사 PC(본인)에서만** 열립니다.

| 화면 | 주소 | 용도 |
|---|---|---|
| **교사 화면** | `/host` | 로비·문제·순위·시상대 진행 (TV/프로젝터 표출) |
| **학생 화면** | `/` (QR 목적지) | 닉네임 입장 → 보기 선택 |
| **퀴즈 만들기** | `/author` | 퀴즈·문항·그림 편집 (교사 PC 전용) |
| **결과** | `/results` | 지난 게임 결과 보기·내보내기 (교사 PC 전용) |

- 화면 좌상단 **teaveloper 엠블럼**을 누르면 언제나 **처음(교사 메인)** 으로 돌아갑니다.
- 우상단 **🌙/☀️ 버튼**으로 라이트/다크 테마를 전환합니다(자동으로 기기 설정도 따름).

## 4. 서버 실행

- `classroom-quiz.exe`를 더블클릭하면 서버가 **모든 네트워크(0.0.0.0)** 에 바인딩됩니다 — 학생이 LAN으로 붙으려면 필수입니다.
- 기본 포트는 **8080**, 사용 중이면 다음 비어 있는 포트를 자동으로 잡습니다.
- 교사 화면에는 **연결 방식 배지**(무선 / 유선)가 표시됩니다. → [10. 네트워크](#10-네트워크-무선--유선)
- 콘솔 창이 보이는 빌드는 **창을 닫으면 서버가 종료**됩니다. 진단용 로그는 exe 옆 `classroom-quiz.log`에 남습니다.

> **방화벽**: 처음 실행 때 Windows가 네트워크 허용을 물으면 **허용**하세요(개인/Private 네트워크). 거부하면 학생이 접속하지 못합니다.

## 5. 퀴즈 만들기 (편집기)

**＋ 퀴즈 만들기**(`/author`)에서 만듭니다. 왼쪽은 퀴즈 목록, 오른쪽은 편집기입니다.

1. **＋ 새 퀴즈** → 위에 **퀴즈 제목**을 입력합니다.
2. 문항마다:
   - **질문**을 입력합니다.
   - (선택) **🖼 그림 추가** → 이미지를 고르면 자동으로 줄여 문항에 넣습니다. (제거도 가능)
   - **보기**를 2~6개 입력하고, 정답인 보기에 **정답** 체크(정답은 1개 이상).
   - **제한시간(초)** 과 **배점**을 정합니다(기본 20초 / 1000점).
   - **＋ 보기 추가**, **＋ 문항 추가**로 늘립니다.
3. **저장**을 누릅니다. (제목·문항·정답이 비어 있으면 안내가 뜹니다.)

- 보기 표시는 **teaveloper 테마 색 + 보기 순번만큼의 엠블럼**(1개·2개 나란히·3개부터 방사형)입니다.
  학생은 **보기 글**을 읽고 고르므로, 엠블럼은 화면을 꾸미는 장식입니다.
- **그림은 별도 파일이 아니라** 퀴즈 JSON 안에 통째로 저장됩니다 → [11. 데이터 저장](#11-데이터-저장과-백업)

## 6. 퀴즈 공유 — 가져오기 / 내보내기

퀴즈 한 개 = **JSON 파일 하나**(그림까지 내장)라 파일만 주고받으면 됩니다.

- **내보내기**: 편집기 위쪽 **↧ 내보내기** → `제목.json` 파일로 저장됩니다(그림 포함).
- **가져오기**: 왼쪽 **↥ 가져오기 (.json)** → 파일을 고르면 새 퀴즈로 목록에 추가됩니다.
  - 가져온 퀴즈는 **항상 새 ID**로 저장되어 **기존 퀴즈를 덮어쓰지 않습니다**.

> 다른 교사에게 줄 때는 내보낸 `.json` 하나만 건네면 됩니다. 받는 사람 PC에 그림 파일이 없어도 문제 그림이 그대로 나옵니다.

## 7. 게임 진행

교사 화면(`/host`)에서 진행합니다.

1. **로비** — QR·접속 주소·참가자 목록이 뜹니다. 퀴즈를 고르고 **시작하기**.
   - **전체화면으로 크게**(또는 QR 클릭): QR을 화면 가득 키웁니다. 옆에 **참가자 명단·인원**이 함께 떠서, 충분히 들어왔는지 보고 닫을 수 있습니다. (아무 곳이나 클릭 또는 **ESC** 로 닫기)
2. **문제** — 질문·보기·타이머가 표시됩니다. 연결된 학생이 **전원 답하면 즉시 공개**, 아니면 시간이 끝나면 공개. **정답 공개 →** 로 넘길 수도 있습니다.
3. **정답 공개** — 정답 표시 + 보기별 응답 분포 막대.
4. **순위표** — 누적 점수 순위 + 등수 변동(▲▼). **다음 문제 →**.
5. **시상대** — Top 3 단상 + 전체 순위. 학생 화면엔 본인 최종 등수가 뜹니다.
6. **처음으로 ↩** 또는 **엠블럼 클릭**으로 로비로 돌아갑니다.

> **재접속**: 학생 페이지는 닉네임을 기억해, 교실 와이파이가 잠깐 끊겨도 **같은 점수로 자동 재접속**됩니다.

---

## 8. 점수 규칙

- **속도 점수**: `배점 × (1 − f/2)`, `f = 응답시간 ÷ 제한시간`. → 빠를수록 만점, 막판에 맞히면 절반.
  타이밍은 **서버 기준**(문제 송출 시각 vs 답안 도착 시각)이라 학생 시계를 신뢰하지 않습니다.
- **연속 정답(스트릭) 보너스**: 연속으로 맞힐 때마다 **+100**, 최대 **+500**.
- 틀리거나 못 맞히면 스트릭은 0으로 초기화됩니다.

## 9. 결과 저장과 내보내기

게임을 **시상대까지 정상 종료**하면 결과가 자동으로 저장됩니다.

- **결과 화면**(`/results`): 퀴즈별로 묶여 **날짜순**으로 쌓입니다. 한 줄을 펼치면 **순위·이름·점수·정답수** 표가 나옵니다.
- **내보내기**:
  - **CSV** — 엑셀로 여는 형성평가 기록용(한글 깨짐 방지 처리됨).
  - **JSON** — 원본 그대로 보관·공유용.
- 필요 없으면 결과별 **삭제**도 가능합니다.

> 게임 중간에 엠블럼으로 로비에 돌아가면(미완료) 결과는 저장되지 않습니다.

## 10. 네트워크 (무선 / 유선)

교사 화면 헤더에 **학생 접속 주소가 어떤 연결인지** 배지로 표시됩니다.

- **무선 (Wi-Fi)** — 학생 무선망에 붙은 일반적인 상태.
- **유선 (랜선)** — 랜선으로 잡힌 상태. 학생 무선망을 직접 구성한 경우가 아니면, 학생이 **같은 와이파이**에 있는지 확인하세요.

설계 메모:
- 학생이 LAN으로 붙어야 하므로 서버는 **전체 인터페이스(0.0.0.0)** 에 바인딩합니다.
- 유선+무선이 섞이면 LAN IP 후보가 여러 개일 수 있어, 사설망 우선순위로 대표 IP를 고릅니다. QR이 안 되면 교사 화면의 접속 주소를 학생에게 직접 불러 주세요.

## 11. 데이터 저장과 백업

모든 데이터는 **exe 옆 폴더**에 파일로 저장됩니다(DB 없음).

```
classroom-quiz.exe
├─ quizzes/<퀴즈id>.json    ← 퀴즈 (문항·보기·그림이 한 파일에)
└─ results/<퀴즈id>/<날짜_시각>.json   ← 게임 결과 (퀴즈별·날짜별)
```

- **그림은 별도 파일이 아닙니다.** 퀴즈 JSON 안에 base64 문자열로 통째로 들어갑니다 → 파일 하나만 옮기면 그림째 따라갑니다.
- 저장은 임시파일→교체 방식이라 저장 중 꺼져도 파일이 깨지지 않습니다.

**백업**: `quizzes/`와 `results/` 폴더를 통째로 복사해 두면 됩니다. 다른 PC로 옮길 때도 이 폴더만 가져가면 그대로 복원됩니다.

## 12. 자주 묻는 질문

**Q. 설치가 필요한가요?**
아니요. `classroom-quiz.exe` 하나만 더블클릭하면 됩니다. 학생도 설치 없이 브라우저로 들어옵니다.

**Q. 인터넷이 안 되는 교실에서도 되나요?**
됩니다. 같은 와이파이(공유기/AP)에만 붙어 있으면 인터넷 없이 동작합니다. 웹 화면과 QR이 exe에 내장돼 있습니다.

**Q. 학생이 QR을 못 찍어요.**
교사 화면의 **접속 주소**(예: `http://192.168.0.x:8080/`)를 불러 주고 학생이 브라우저에 직접 입력하게 하세요. QR을 **전체화면으로 크게** 띄우면 멀리서도 잘 찍힙니다.

**Q. 학생이 안 보여요 / 접속이 안 돼요.**
① 학생이 **교사 PC와 같은 와이파이**인지, ② 헤더 배지가 **유선**이면 학생 무선망과 다른 망일 수 있으니 확인, ③ 처음 실행 때 **방화벽 허용**을 눌렀는지 확인하세요.

**Q. 그림은 어디에 저장되나요?**
퀴즈 JSON 파일 **안에** 저장됩니다(경로가 아니라 그림 자체). 그래서 `quizzes/` 폴더엔 `.json`만 보이고, 파일 하나만 공유하면 그림도 함께 전달됩니다.

**Q. 다른 선생님이 만든 퀴즈를 쓰려면?**
받은 `.json` 파일을 **↥ 가져오기**로 넣으면 됩니다. 항상 새 퀴즈로 추가되어 내 퀴즈를 덮지 않습니다.

**Q. 결과를 엑셀로 보고 싶어요.**
**📊 결과** → 해당 게임 펼치기 → **CSV 내보내기**. 엑셀에서 바로 열리고 한글도 깨지지 않습니다.

**Q. 진행 중에 처음으로 돌아가려면?**
왼쪽 위 **엠블럼**을 누르세요. 진행 중이면 확인 후 로비로 돌아갑니다.
